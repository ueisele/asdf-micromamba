# asdf-micromamba

[Micromamba](https://mamba.readthedocs.io/) plugin for the [asdf version manager](https://asdf-vm.com).

# Install

```bash
asdf plugin add micromamba https://github.com/ueisele/asdf-micromamba.git
```
